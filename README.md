## Game Proposal ##

I want to extend breakout by adding power-ups to certain "lucky" blocks (different color from regular blocks) and then having a final level (most likely level 5) where you have to use your power-ups to clear all the blocks and win the game. I will also have some "evil" blocks (different color from regular blocks) where if you hit them then you have to restart the current level.

**Game Controls:**

+ Left and Right arrow keys control paddle movement
+ Space Key controls power-up use

**Game Elements:**

+ Blocks
+ Ball
+ Power-ups
+ Paddle

**How to Win:**

+ Clear all the blocks on level 5 using power-ups
